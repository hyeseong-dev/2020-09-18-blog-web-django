# 소개
* 파이썬의 성장에 힘입어 장고Django 역시 매우 빠르게 사용자층이 넓어지고 있습니다. 파이썬이 언어를 배우는 것이라면, 
장고는 바로 결과물을 만들 수 있는 웹 프레임워크입니다. 많은 래퍼런스도 있고 여러 필요한 오픈소스 라이브러리 역시 
풍부합니다. 우리에게 알려진 Instagram, Pinterest 서비스 역시 장고로 개발되었으며, 최근 구글 트랜드를 봐도 장고의 인기는 다른 프레임워크에 비해
월등히 높은 것을 알수 있습니다. 이에 Django를 이용한 프로젝트를 진행해 보았습니다.

## 설치방법
* > git clone https://github.com/hyeseong-dev/2020-09-18-blog-web-django.git
* > pip install Django Pillow django-crispy-forms
* > python manage.py makemigrations
* > python manage.py migrate
* > python manage.py runserver

## 기능 
1. 유저가 등록한 글을 메인 화면 or 유저별로 분류된 화면에 글을 보여줌
2. 로그인, 로그아웃, 회원가입, 비밀번호 리셋
3. 개인정보, 개인정보 수정(이메일, 프로파일 이미지)
4. 페이지내이션
5. 글 포스팅, 글 수정, 글 삭제
